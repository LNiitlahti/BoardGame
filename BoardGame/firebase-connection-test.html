How to Get Your Firebase Config

  1. Go to the Firebase Console
  2. Select your project
  3. Click the gear icon (⚙️) next to "Project Overview" to open Project settings
  4. Scroll down to "Your apps" section
  5. Select your web app or create a new one with the "Add app" button
  6. In the "SDK setup and configuration" section, you'll find your Firebase config object
  7. Copy the entire configuration object and paste it above
